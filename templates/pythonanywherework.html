07 : PythonAnywhere

PythonAnywhere is an online integrated development environment (IDE) and web hosting service (Platform as a service) based on the Python programming language.

Steps to host a website

1. Create a Beginner account in PythonAnywhere
Head over to www.pythonanywhere.com/pricing and click on the button that says “Create a Beginner account.” This will allow you to create a free account. After clicking that, you will be asked to enter your email and to create a username and password. Make sure you choose your username wisely, because you will NOT be able to change it and because your website domain will be https://www.your-username.pythonanywhere.com. Make sure to agree to their “Terms and Conditions,” then click register.

2. Clone your repository
  1. Open the Bash Console
  2. Type 
    git clone yourGitLink
    and press enter.

3. Creating virtual environment and install Flask
Go to the Consoles tab and start a Bash console. Specify which Python version to use for your virtualenv using the --python option, but note that it must match the version of Python you've chosen for your web app. So, to create a new Python 3.6 virtualenv, run this command:: 
mkvirtualenv myvirtualenv --python=/usr/bin/python3.9
Then activate the virtual environment by executing the command
workon myvirtualenv
Install Flask 
pip install flask
or 
pip install -r requirements.txt
4. Hosting the website
  1. Near the top right corner of your dashboard, click on the “Web” link. To create a new web app, click on the blue “Add a new web app” button.
  2. Now choose the python version we choose 3.9
  3. Next select Manual configuration
  4. In the 'code' section change source path as 'Portfolio'
  5. In the 'code' section click on 'WSGI configuration file' link and change the code in it to 
    import sys
    path = '/home/`your_username`/Portfolio'
    if path not in sys.path:
      sys.path.append(path)
    from server import app as application
  6. Save it and go back
  7. In the 'Virtualenv' section set it to 'myvirtualenv'
  8. And done! go to the top of the page click on Reload and go to your page
