06 : Github

GitHub is the best place to share code with friends, co-workers, classmates, and complete strangers. Many people use GitHub to build amazing things together.

Steps to upload website source on Github


1. Create a repository
You can store a variety of projects in GitHub repositories, including open source projects. With open source projects, you can share code to make better, more reliable software. You can use repositories to collaborate with others and track your work. For more information, see "About repositories."
  1. In the upper-right corner of any page, use the + drop-down menu, and select New repository.
  2. Give repository name, we have given Portfolio
  3. Choose a repository visibility as public so that you can clone the repo when hosting it.
  4. Select Initialize this repository with a README.
  5. Click Create repository.

2. Adding a file to a repository on GitHub
  1. On GitHub, navigate to the main page of the repository.
  2. Above the list of files, using the Add file drop-down, click Upload files.
  3. Drag and drop the file or folder you'd like to upload to your repository onto the file tree.
  4. At the bottom of the page, type a short, meaningful commit message that describes the change you made to the file.
  5. Click Commit changes.
